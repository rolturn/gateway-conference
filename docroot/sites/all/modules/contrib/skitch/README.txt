About
-----------
Filters Skitch links in content and replaces with Skitch images

Installation
------------
  1. Download the Simple HTML DOM Parser (http://simplehtmldom.sourceforge.net/) and extract the file under sites/all/libraries.
  2. Download and enable this module.
  3. Configure at Administer > Configuration > Content Authoring > Text formats

Dependencies
-----------
  1. Libraries API
